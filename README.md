##  Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript

- **Animations:** CSS Animations, Intersection Observer API
- **Tools:** Git, GitHub

- Scroll reveal animations
- Fixed navigation bar
- Resume download
- External links to GitHub & LinkedIn
